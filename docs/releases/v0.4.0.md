# Triluga v0.4.0

## Destaques

- reposicionamento do produto para uma experiência mais próxima de classificados de aluguel
- catálogo público acessível sem exigir login para busca e detalhe de anúncio
- fluxo de publicação simplificado, com menos recursos legados e orientação mais direta para publicar anúncios
- menu, busca, chat e detalhe do anúncio revisados para reduzir atrito e melhorar clareza

## Entregas principais

- melhoria do cadastro com placeholders, confirmação de senha e visualização da senha
- remoção da feature antiga de `my-bookings` do fluxo principal
- redirecionamento do antigo `owner-dashboard` para a área de publicação
- ajustes de copy e navegação entre home, busca, anúncio, chat e publicar
- limpeza profunda de código legado na área do anunciante

## Observações

- a build do frontend segue com warnings de budget CSS já conhecidos, mas a aplicação compila normalmente
