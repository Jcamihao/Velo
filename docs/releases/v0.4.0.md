# Release Notes v0.4.0 - Triluga Analytics & Carrousel

## O que há de novo

Esta versão foca em melhorar o engajamento do usuário através de um sistema de analytics robusto e um carrossel de anúncios dinâmico na página inicial. Além disso, realizamos uma grande refatoração na estrutura do frontend para garantir melhor escalabilidade do projeto.

### Principais Funcionalidades

- **Rastreamento de Visualizações (Analytics)**:
  - Novo modelo `VehicleView` no banco de dados para registrar cada visualização de veículo.
  - Endpoints no backend para registrar visualizações e recuperar os veículos mais populares.
  - Serviço de monitoramento no frontend integrado aos componentes de detalhe de veículo.

- **Carrossel de Anúncios Dinâmico**:
  - Implementação de um carrossel na home page que destaca automaticamente os veículos com maior número de visualizações.
  - Design premium com micro-animações e suporte a gestos de deslize.

- **Refatoração Estrutural (Frontend)**:
  - Migração de componentes de arquivos únicos para diretórios organizados (TS, HTML, SCSS separados).
  - Limpeza e organização dos módulos de `shared` e `features`.

### Mudanças Técnicas

- **Backend**:
  - Adição de `VehicleView` ao Prisma Schema.
  - Implementação do `AnalyticsModule`, `AnalyticsService` e `AnalyticsController`.
  - Atualização do `VehiclesService` para integração com os dados de popularidade.
- **Frontend**:
  - Criação do `AnalyticsTrackingService`.
  - Reorganização de mais de 80 arquivos de componentes.
  - Otimização de estilos globais e inclusão de novos assets de branding.
