# Release Notes v0.5.0 - Triluga Classificados

Data: `2026-04-12`

## Resumo

Esta versão transforma o Triluga em um classificado de veículos entre pessoas. O fluxo transacional antigo foi removido do produto ativo para concentrar a experiência em descoberta, anúncio, favoritos, chat, reputação e administração.

## Entregas principais

- remoção do domínio de reserva, disponibilidade, bloqueios, pricing preview e pagamentos do backend
- limpeza do schema Prisma, seed e migration para remover tabelas, enums e colunas transacionais
- remoção das telas e serviços de solicitação de reserva, minhas reservas e checklist
- ajuste da busca, detalhe do veículo, cards, perfil público, admin e privacidade para linguagem de classificado
- CTA principal do detalhe direcionado para conversa com o anunciante
- remoção do ícone de fechar no card de localização da página de detalhe

## Impacto esperado

- produto mais simples e alinhado ao posicionamento de classificados
- menos superfície de manutenção em backend, frontend e banco
- navegação pública focada em encontrar veículos e iniciar contato
- documentação e QA atualizados para o novo escopo

## Validação

- `npm --prefix backend run prisma:generate`
- `npx prisma validate --schema prisma/schema.prisma`
- `npm run build`
- `npm --prefix backend test -- --runInBand`
- `npx tsc --noEmit -p tsconfig.json`
