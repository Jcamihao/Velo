# Release Notes v0.7.0 - Triluga Home, Busca e Perfis

Data: `2026-04-23`

## Resumo

Esta versão consolida a navegação da home e da busca com foco em volume de anúncios, sticky header, perfil público mais completo e ambiente local consistente em `localhost:3002`.

## Entregas principais

- home com remoção da seção "Como funciona" e troca da vitrine de marcas por categorias
- listagem de carros da home com carregamento incremental no mobile e paginação local no desktop
- correção da busca para continuar carregando resultados além da primeira página
- header de busca compartilhado com comportamento sticky refinado para home e tela de busca
- reorganização do topo da busca com filtros e notificações no header sticky
- perfil público com skeleton loading, badge de tempo de resposta, ações rápidas e novas métricas de confiança
- alinhamento dos defaults locais para backend em `http://localhost:3002`
- script `db:add:cars` para popular o banco com quantidade arbitrária de carros aleatórios

## Impacto esperado

- home mais direta, com navegação por categorias e melhor leitura em mobile
- busca mais confiável para bases maiores, sem travar na primeira página de resultados
- cabeçalho mais consistente entre home e busca durante o scroll
- perfil público com mais sinais de confiança e ações úteis para contato e compartilhamento
- setup local menos sujeito a inconsistências entre frontend, backend e base de dados

## Validação

- `npm --prefix backend run verify`
- `npm --prefix frontend run build`
